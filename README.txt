How to use on iPad:
1. Unzip this folder.
2. Open index.html in Safari if your file app allows it.
3. Best method: upload this folder to any simple web hosting / iCloud static host / local server, open the link in Safari, then tap Share > Add to Home Screen.
4. Prices are saved on the iPad browser using local storage.
5. Receipt is formatted for small thermal roll printing.
6. Phone number is not shown.

Note:
This is a PWA/web-app version, which is the best practical iPad option without Mac/Xcode/App Store.
